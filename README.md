# 한국어 단어 데이터베이스
본 단어 데이터베이스는 국립국어원 표준대사전의 OpenAPI를 사용해 긁어온 단어를 DB화 시킨 것입니다.
## 구성
본 데이터베이스의 테이블에는 컬럼 두 개 외에는 아무것도 없습니다. 테이블의 구성은 다음 표와 같습니다.
|컬럼|설명|
|--|--|
|ID|정수 형식의 식별용 아이디입니다.|
|WORD|문자열 형식의 단어입니다.|
## 작동 환경
|DB 엔진|작동 여부|
|--|--|
|MariaDB 10.5.8|✔️|
|MySQL 8.0.23|✔️|
